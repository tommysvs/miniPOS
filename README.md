# miniPOS

## General Description
miniPOS is a Point of Sale (POS) system built using Windows Forms and .NET 8. 

The system consists of six main screens that work together to provide a seamless management experience:
- **Login**: Handles user authentication to secure the application.
- **Dashboard**: Provides a general overview and quick access to operations.
- **Categories**: Allows you to manage product categories for better organization.
- **Products**: Manages the product inventory.
- **Clients**: Manages customer information and records.
- **Suppliers**: Manages supplier information and relationships.

Each screen serves a specific purpose, and they all share a unified navigation menu, allowing users to move smoothly between different sections of the application.

## Technologies Used
- **Language**: C#
- **UI Architecture**: Windows Forms (WinForms)
- **Framework**: .NET 8.0 SDK or later
- **IDE**: Visual Studio 2022 (recommended)

## Features
- **Centralized Navigation**: A shared navigation menu across all screens ensures a consistent user experience.
- **Data Management**: Dedicated screens for handling products, categories, clients, and suppliers.
- **Secure Access**: A built-in login screen to protect sensitive business data.
